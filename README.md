This landing page is a simple, static website designed to present key information in a clean and accessible way. It is built using standard web technologies and deployed automatically through GitHub Pages.

## 🛠️ Technologies Used
- HTML5  
- CSS3

## 📚 How to Update the Site
1. Make changes to your files (`index.html`, `index.css`, etc.)
2. Commit and push to the `main` branch
3. GitHub Pages will automatically rebuild and update the live site
